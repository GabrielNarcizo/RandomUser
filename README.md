# RANDOMUSER CASEIRA 🏡
## _Criar uma **API** que retorna informações de um usuário_ 💻

##### DESAFIO 🙀

_As informações dos usuários devem conter, pelo menos:_

- Primeiro nome
- Segundo nome
- Gênero
- Data de nascimento
- Email
- Senha

**Tecnologias** 🔧
- Node.js
- Express
- Faker
